# CleanMyCodeMac

A macOS disk cleanup tool built for developers. Deeply cleans caches and redundant files generated by development environments to free up disk space fast.

[中文文档](README_ZH.md)

## Features

- **System Cache** — Clean temporary caches from macOS system apps
- **App Cache** — Clean caches from Chrome, VSCode, JetBrains, and other third-party apps
- **Dev Cache** — Clean caches from Node.js, Python, Ruby, Rust, Go, Java, and other toolchains
- **Documents** — Scan PDF, Word, Excel, Markdown, iWork, and other document files
- **Media** — Scan images, audio, and video files
- **Log Files** — Clean crash reports and runtime logs older than 7 days
- **Downloads** — Analyze old files in the Downloads folder, grouped by type
- **Large Files** — Find files larger than 500MB with deep analysis (including Docker disk usage)
- **Trash** — Empty the Trash with one click

## Highlights

- Native macOS window (pywebview + WKWebView), not a browser
- Local-first: scans and cleanup decisions run on your Mac
- Grouped scan results with expand-by-category/app drill-down
- Safety labels (Safe to clean / Use caution)
- Files moved to Trash — recoverable
- Quick reveal in Finder
- Configurable scan scope

## Quick Start

### Requirements

- macOS 12.0+
- Python 3.10+

### Install Dependencies

```bash
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

### Run

```bash
# Option 1: Command line
cd cleanmycodemac
../venv/bin/python3 main.py

# Option 2: Double-click run.command
```

### Build as .app

```bash
pip install -r requirements-build.txt
python3 -m PyInstaller --noconfirm CleanMyCodeMac.spec
# Output: dist/CleanMyCodeMac.app
```

See [BUILD.md](BUILD.md) for full build instructions.

## Privacy & Permissions

- All scan and cleanup logic runs locally on your Mac.
- No cloud service is required for normal use.
- Some scan targets may require Full Disk Access to be fully visible.
- Cleanup operations move files to Trash when possible, so items remain recoverable.

## Configuration

A config file is created at `~/.cleanmycodemac_config.json` on first run:

| Key | Default | Description |
|-----|---------|-------------|
| `large_file_threshold_mb` | 500 | Large file threshold (MB) |
| `old_download_days` | 30 | Downloads expiry (days) |
| `old_log_days` | 7 | Log expiry (days) |
